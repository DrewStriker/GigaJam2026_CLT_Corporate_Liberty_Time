Basic Font Information

"Amorria" Font is a Unicode typeface family that supports all languages
that use the Latin script and its variants, and could be expanded to support
other scripts.


"Amorria" Font you'll find at: https://glottastudio.com

"Amorria" Font is created with FontCreator.


More specifically, this release supports the following Unicode ranges:

 - Basic Latin and Controls 
 - General Punctuation 
 - Currency Symbols 
 - Mathematical Operators 

This release contains:

 - Amorria.ttf
 - Amorria.otf
 - Amorria.woff
 - SIL - OpenFont License.txt - official license text
